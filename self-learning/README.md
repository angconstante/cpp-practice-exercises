# Self-learning

This forlder contains independent 
programming study, practice exercises and 
personal projects completed outside of CS Coursework. 

The purpose of this sections is have hands-on learning
and practicing programming skills in order to develop
and strengthen software development fundumentals. 

## Categories

### Practice Projects
Exercises and tutorials that reinforce learned skills through coursework
and learn new concepts. Serves as independent practice.

### Personal Projects
Original projects built to practice and apply all learned concepts.

### Skill Development
Exercises that help improve technical skills. 
A problem-solving/debugging folder.

### Independent Study
Self-directed, misc concepts of Computer Science. 
Topics beyond formal Coursework
